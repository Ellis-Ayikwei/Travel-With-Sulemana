# @Campus Ride/shared

Shared TypeScript types, constants, and utilities for the Campus Ride platform.

## Usage

### In Next.js Frontend
```typescript
import { User, Errand, TRUST_BADGES } from "@Campus Ride/shared";
```

### In Node.js Backend
```typescript
import { JWTPayload, ERRAND_STATUS } from "@Campus Ride/shared";
```

## Building
```bash
npm run build
```
